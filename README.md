## ✨ Features

- 🚀 **Pure Frontend** - No server, no data transmission, 100% privacy
- 📝 **Full Markdown Support** - Headers, lists, code blocks, tables, quotes
- 🎨 **4 Theme Styles** - Default, Dark, Green, Blue
- 📋 **One-Click Copy** - Direct paste to WeChat editor
- 📱 **Responsive** - Works on mobile and desktop
- ⚡ **Real-time Preview** - See changes instantly

## 🎯 Use Cases

- Technical blog posts to WeChat articles
- Documentation formatting
- Any Markdown to WeChat conversion needs

## 🚀 Usage

1. Visit [https://walkmanhenry.github.io/md2wechat](https://walkmanhenry.github.io/md2wechat)
2. Paste your Markdown in the left panel
3. Preview the formatted result in the right panel
4. Click "Copy HTML" and paste into WeChat editor

## 🔒 Privacy

All processing happens in your browser. **No data is sent to any server.** Your content never leaves your device.

## 🛠️ Development

```bash
git clone https://github.com/WalkmanHenry/md2wechat.git
cd md2wechat
# Open index.html in your browser
```

## 📝 Supported Markdown Syntax

| Syntax | Result |
|--------|--------|
| `# Heading` | H1 Header |
| `## Heading` | H2 Header |
| `**bold**` | **Bold** |
| `*italic*` | *Italic* |
| `` `code` `` | Inline code |
| ```` ```code block``` ```` | Code block |
| `> quote` | Blockquote |
| `- item` | Unordered list |
| `1. item` | Ordered list |
| `\|a\|b\|` | Table |
| `---` | Horizontal rule |
