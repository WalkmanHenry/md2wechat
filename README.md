# MD2WeChat - Markdown转微信排版

一个免费的在线工具，将 Markdown 转换为微信公众账号兼容的 HTML 格式。

## ✨ 特性

- 🚀 **纯前端运行** - 无需服务器，保护隐私
- 📝 **完整 Markdown 支持** - 标题、列表、代码、表格等
- 🎨 **多主题预览** - 默认、深色、绿色、蓝色主题
- 📋 **一键复制** - 直接粘贴到微信编辑器
- 📱 **响应式设计** - 支持移动端使用

## 🎯 使用场景

- 微信公众号文章排版
- 技术博客转微信文章
- 快速格式化文档
